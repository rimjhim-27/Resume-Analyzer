# 📌 Overview

The **AI Resume Analyzer** is an intelligent web application that helps job seekers evaluate and improve their resumes. It extracts information from uploaded PDF resumes, calculates an ATS (Applicant Tracking System) compatibility score, identifies technical skills, classifies resumes into professional domains, and provides personalized suggestions to increase the chances of getting shortlisted.

Whether you're a student, fresher, or experienced professional, this tool helps optimize your resume for modern recruitment systems.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/rimjhim-27/Resume-Analyzer.git
```

## Move to Project Directory

```bash
cd Resume-Analyzer
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```

The application will start at:

```
http://localhost:8501
```

# 🚀 How It Works

### Step 1

Upload your resume in **PDF** format.

↓

### Step 2

The application extracts text from your resume.

↓

### Step 3

Machine Learning and NLP techniques analyze the content.

↓

### Step 4

Technical skills are identified automatically.

↓

### Step 5

The resume is classified into an appropriate professional domain.

↓

### Step 6

An ATS Compatibility Score is calculated.

↓

Personalized recommendations and visual insights are displayed.
